# run-forrest-run
Jogo 2D desenvolvido em HTML5, JavaScript e CSS com três níveis (ping-pong, pesca e corrida). Projeto académico inspiado no filme Forrest Gump.
Trabalho desenvido no âmbito da unidade curricular Desenvolvimento de Aplicações Multimédia, do curso Comunicação e Design Multimédia, da Escola Superior de Educação, no ano letivo 2024/2025.
